# Student Loan Evaluator

A web-based tool to evaluate the worthiness of different student loans and provide AI-driven financial advice.

## Features

- **Loan Options:**
  - Private Loans
  - Public Subsidized Loans
  - Public Unsubsidized Loans

- **Input Parameters:**
  - Interest Rate
  - Repayment Period
  - Expected Salary (Hourly/Annual)
  - Current Debt
  - Future Borrowing Needs

- **Output Calculations:**
  - Total Cost of Loan
  - Monthly Repayment Amount
  - Total Repayment Amount
  - Long-term Financial Impact

- **AI Financial Advice:**
  - Recommendations based on input data
  - Alternative loan suggestions
  - Potential savings plans
